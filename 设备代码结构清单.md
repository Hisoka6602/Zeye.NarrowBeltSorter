## ZhiQian（智嵌）

智嵌设备用于格口控制，主要通过 TCP 通信实现 32 路继电器状态读写与批量控制。

```text
ZhiQian
├── Drivers
│   └── Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian  # 智嵌设备驱动目录
│       ├── ZhiQianBinaryClientAdapter.cs         # 智嵌设备 TCP 客户端适配器（指令收发、重试、连接管理）
│       ├── ZhiQianChuteManager.cs                # 智嵌格口管理器
│       ├── ZhiQianChute.cs                       # 智嵌格口实体实现
│       └── ZhiQianClientAdapterFactory.cs        # 智嵌客户端适配器工厂默认实现
├── Core
│   ├── Zeye.NarrowBeltSorter.Core/Manager/Chutes/IZhiQianClientAdapter.cs      # 智嵌协议无关客户端接口
│   ├── Zeye.NarrowBeltSorter.Core/Manager/Chutes/IZhiQianClientAdapterFactory.cs # 智嵌客户端适配器工厂接口
│   ├── Zeye.NarrowBeltSorter.Core/Options/Chutes/ZhiQianChuteOptions.cs         # 智嵌格口聚合配置
│   ├── Zeye.NarrowBeltSorter.Core/Options/Chutes/ZhiQianDeviceOptions.cs        # 智嵌单设备配置
│   ├── Zeye.NarrowBeltSorter.Core/Options/Chutes/ZhiQianLoggingOptions.cs       # 智嵌日志配置
│   └── Zeye.NarrowBeltSorter.Core/Utilities/Chutes/ZhiQianAddressMap.cs         # 智嵌地址映射工具
└── 文档
    └── Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian/doc  # 智嵌设备文档目录
        ├── 智嵌32路网络继电器手册解析与IChuteManager接入方案.md  # 智嵌设备协议解析与接入设计文档
        └── 【智嵌物联】32路网络继电器控制器用户使用手册V1.2.pdf  # 智嵌设备官方用户手册
```

│   ├── Zeye.NarrowBeltSorter.Core/Enums/Emc/EmcControllerStatus.cs  # 雷赛 EMC 状态枚举
│   ├── Zeye.NarrowBeltSorter.Core/Enums/Io/IoPanelMonitoringStatus.cs  # IoPanel 监控状态枚举

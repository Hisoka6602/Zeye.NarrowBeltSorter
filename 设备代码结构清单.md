# 设备代码结构清单

## 维护约束

- 本文档按“设备”分章节维护代码结构。

## ZhiQian（智嵌）

智嵌设备用于格口控制，主要通过 TCP 通信实现 32 路继电器状态读写与批量控制。

### Drivers

- `Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian/ZhiQianBinaryClientAdapter.cs`：智嵌设备 TCP 客户端适配器（指令收发、重试、连接管理）。
- `Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian/ZhiQianChuteManager.cs`：智嵌格口管理器。
- `Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian/ZhiQianChute.cs`：智嵌格口实体实现。
- `Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian/IZhiQianClientAdapterFactory.cs`：智嵌客户端适配器工厂接口。
- `Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian/ZhiQianClientAdapterFactory.cs`：智嵌客户端适配器工厂默认实现。

### Core / Options / Manager / Utilities

- `Zeye.NarrowBeltSorter.Core/Manager/Chutes/IZhiQianClientAdapter.cs`：智嵌协议无关客户端接口。
- `Zeye.NarrowBeltSorter.Core/Options/Chutes/ZhiQianChuteOptions.cs`：智嵌格口聚合配置。
- `Zeye.NarrowBeltSorter.Core/Options/Chutes/ZhiQianDeviceOptions.cs`：智嵌单设备配置。
- `Zeye.NarrowBeltSorter.Core/Options/Chutes/ZhiQianLoggingOptions.cs`：智嵌日志配置。
- `Zeye.NarrowBeltSorter.Core/Utilities/Chutes/ZhiQianAddressMap.cs`：智嵌地址映射工具。

### 文档

- `Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian/doc/智嵌32路网络继电器手册解析与IChuteManager接入方案.md`
- `Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian/doc/【智嵌物联】32路网络继电器控制器用户使用手册V1.2.pdf`

## LeiMa（雷码）

雷码设备用于环道驱动与调速控制，核心通信链路为 Modbus（含 TCP/RTU 场景）。

### Drivers

- `Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/LeiMaModbusClientAdapter.cs`：雷码 Modbus 客户端适配器。
- `Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/LeiMaLoopTrackManager.cs`：雷码环道管理器。
- `Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/LeiMaSerialRtuSharedConnection.cs`：雷码串口 RTU 共享连接上下文。

### Core / Options

- `Zeye.NarrowBeltSorter.Core/Options/LoopTrack/LoopTrackLeiMaConnectionOptions.cs`：雷码连接参数配置。

### 文档

- `Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/doc/雷码LM1000H说明书参数与调用逻辑梳理.md`
- `Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/doc/雷码快速调机参数变频器配置表梳理.md`
- `Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/doc/多从站稳速难题分析与工程解决方案.md`
- `Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/doc/雷码Modbus读写超时根因分析.md`
- `Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/doc/2-LM1000H 说明书.pdf`
- `Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/doc/(雷码)快速调机参数20250826.xlsx`

## Leadshaine（雷赛）

雷赛设备用于红外驱动与运动控制场景，包含红外驱动器帧编解码与 EMC 控制接口。

### Drivers / Infrared

- `Zeye.NarrowBeltSorter.Drivers/Vendors/Leadshaine/Infrared/LeadshaineInfraredDriverFrameCodec.cs`：雷赛红外驱动器帧编解码。

### Core / Manager / Options

- `Zeye.NarrowBeltSorter.Core/Manager/Chutes/IInfraredDriverFrameCodec.cs`：红外驱动器帧编解码接口。
- `Zeye.NarrowBeltSorter.Core/Options/Chutes/InfraredChuteOptions.cs`：红外格口配置。

### 文档

- `Zeye.NarrowBeltSorter.Drivers/Vendors/Leadshaine/Infrared/doc/LeadshaineEmcController完整接入与IO监控步骤.md`
- `Zeye.NarrowBeltSorter.Drivers/Vendors/Leadshaine/Infrared/doc/LDC-FJ-RF红外驱动器接入ICarrierManager详细方案.md`
- `Zeye.NarrowBeltSorter.Drivers/Vendors/Leadshaine/Infrared/LDC-FJ-RF分拣专用红外驱动器使用说明书-20240618.pdf`

### Drivers / Emc

- `Zeye.NarrowBeltSorter.Drivers/Vendors/Leadshaine/Emc/LTDMC.cs`：雷赛运动控制底层互操作封装。
- `Zeye.NarrowBeltSorter.Drivers/Vendors/Leadshaine/Emc/LTDMC.dll`：雷赛运动控制动态库。

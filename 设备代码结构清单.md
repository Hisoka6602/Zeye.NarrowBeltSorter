# 设备代码结构清单

## 维护约束

- 本文档按“设备”分章节维护代码结构。

## ZhiQian（智嵌）

智嵌设备用于格口控制，主要通过 TCP 通信实现 32 路继电器状态读写与批量控制。

```text
ZhiQian
├── Drivers
│   └── Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian  # 智嵌设备驱动目录
│       ├── ZhiQianBinaryClientAdapter.cs         # 智嵌设备 TCP 客户端适配器（指令收发、重试、连接管理）
│       ├── ZhiQianChuteManager.cs                # 智嵌格口管理器
│       ├── ZhiQianChute.cs                       # 智嵌格口实体实现
│       ├── IZhiQianClientAdapterFactory.cs       # 智嵌客户端适配器工厂接口
│       └── ZhiQianClientAdapterFactory.cs        # 智嵌客户端适配器工厂默认实现
├── Core
│   ├── Zeye.NarrowBeltSorter.Core/Manager/Chutes/IZhiQianClientAdapter.cs      # 智嵌协议无关客户端接口
│   ├── Zeye.NarrowBeltSorter.Core/Options/Chutes/ZhiQianChuteOptions.cs         # 智嵌格口聚合配置
│   ├── Zeye.NarrowBeltSorter.Core/Options/Chutes/ZhiQianDeviceOptions.cs        # 智嵌单设备配置
│   ├── Zeye.NarrowBeltSorter.Core/Options/Chutes/ZhiQianLoggingOptions.cs       # 智嵌日志配置
│   └── Zeye.NarrowBeltSorter.Core/Utilities/Chutes/ZhiQianAddressMap.cs         # 智嵌地址映射工具
└── 文档
    └── Zeye.NarrowBeltSorter.Drivers/Vendors/ZhiQian/doc  # 智嵌设备文档目录
        ├── 智嵌32路网络继电器手册解析与IChuteManager接入方案.md  # 智嵌设备协议解析与接入设计文档
        └── 【智嵌物联】32路网络继电器控制器用户使用手册V1.2.pdf  # 智嵌设备官方用户手册
```

## LeiMa（雷码）

```text
LeiMa
├── Drivers
│   └── Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa  # 雷码设备驱动目录
│       ├── LeiMaModbusClientAdapter.cs           # 雷码 Modbus 客户端适配器
│       ├── LeiMaLoopTrackManager.cs              # 雷码环道管理器
│       └── LeiMaSerialRtuSharedConnection.cs     # 雷码串口 RTU 共享连接上下文
├── Core
│   └── Zeye.NarrowBeltSorter.Core/Options/LoopTrack/LoopTrackLeiMaConnectionOptions.cs  # 雷码连接参数配置
└── 文档
    └── Zeye.NarrowBeltSorter.Drivers/Vendors/LeiMa/doc  # 雷码设备文档目录
        ├── 雷码LM1000H说明书参数与调用逻辑梳理.md  # 雷码参数语义与调用映射说明
        ├── 雷码快速调机参数变频器配置表梳理.md  # 雷码现场调机参数整理文档
        ├── 多从站稳速难题分析与工程解决方案.md  # 多从站稳速问题分析与方案对比
        ├── 雷码Modbus读写超时根因分析.md  # 雷码Modbus超时问题排查文档
        ├── 2-LM1000H 说明书.pdf  # 雷码LM1000H官方说明书
        └── (雷码)快速调机参数20250826.xlsx  # 雷码快速调机参数原始表格
```

## Leadshaine（雷赛）

雷赛设备用于红外驱动与运动控制场景，包含红外驱动器帧编解码与 EMC 控制接口。

```text
Leadshaine
├── Drivers
│   └── Zeye.NarrowBeltSorter.Drivers/Vendors/Leadshaine  # 雷赛设备驱动目录
│       ├── Infrared
│       │   └── LeadshaineInfraredDriverFrameCodec.cs      # 雷赛红外驱动器帧编解码
│       └── Emc
│           ├── LTDMC.cs                                    # 雷赛运动控制底层互操作封装
│           └── LTDMC.dll                                   # 雷赛运动控制动态库
├── Core
│   ├── Zeye.NarrowBeltSorter.Core/Manager/Chutes/IInfraredDriverFrameCodec.cs  # 红外驱动器帧编解码接口
│   └── Zeye.NarrowBeltSorter.Core/Options/Chutes/InfraredChuteOptions.cs        # 红外格口配置
└── 文档
    ├── Zeye.NarrowBeltSorter.Drivers/Vendors/Leadshaine/Infrared/doc  # 雷赛红外文档目录
    │   ├── LeadshaineEmcController完整接入与IO监控步骤.md  # 雷赛EMC控制器接入与监控步骤说明
    │   └── LDC-FJ-RF红外驱动器接入ICarrierManager详细方案.md  # 雷赛红外驱动器接入方案文档
    └── Zeye.NarrowBeltSorter.Drivers/Vendors/Leadshaine/Infrared/LDC-FJ-RF分拣专用红外驱动器使用说明书-20240618.pdf  # 雷赛红外驱动器官方说明书
```

## SiemensS7（西门子）

西门子 S7 设备目录当前已落地 PR-1 基础能力（抽象、配置、校验与 Host 注册入口），驱动实现仍待后续 PR 落地。

├── Core

│   ├── Zeye.NarrowBeltSorter.Core/Events/Emc/EmcInitializedEventArgs.cs  # EMC 初始化完成事件载荷
│   ├── Zeye.NarrowBeltSorter.Core/Events/Emc/EmcStatusChangedEventArgs.cs  # EMC 状态变更事件载荷
│   ├── Zeye.NarrowBeltSorter.Core/Events/Emc/EmcFaultedEventArgs.cs  # EMC 故障事件载荷

│   ├── Zeye.NarrowBeltSorter.Core/Enums/SiemensS7/SiemensS7AddressArea.cs  # 西门子 S7 地址区枚举
│   └── Zeye.NarrowBeltSorter.Core/Options/SiemensS7  # 西门子 S7 配置模型目录（连接/点位绑定/传感器/聚合校验）
├── Host

└── Tests
